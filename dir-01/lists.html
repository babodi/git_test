Ordered and Unordered Lists

Lists can be ordered, meaning they ascend numerically, or unordered which are like plain bullet points.

Ordered List

The steps to creating an ordered list are as follows:

  1. Create an ordered list element. (ol)
  2. Nest list items inside list element. (li)
  3. Close list element.
