HTML Boilerplate

This is a document demonstrating basic HTML boilerplate.

DOCTYPE Element

Every HTML page starts with a DOCTYPE tag at beginning of the file.

This ensures that you are using the latest version of HTML.

HTML Element

All other elements reside withing the HTML element.

You could say they are children of the parent element.

Head Element

The head element is where the metadata of the page lives.

Meta Element

You can define the character encoding inside the meta element using the charset attribute. This ensures that special symbols/characters show in the browser correctly.

Title Element

You can change the title of the tab using the title element. Without it, the page will default to it's filename.

Body Element

The body element is the parent to all these headers and paragraphs you see on the screen. Information that is intended to be seen resides here.
